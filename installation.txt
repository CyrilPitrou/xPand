General Information:
*******************

Visit http://www2.iap.fr/users/pitrou/xpand.htm

Installation:
************

The package can be downloaded as a single tar ball xPand_0.0.1.tar.gz.
When uncompressed (typing 'tar -xzf xPand_0.0.1.tar.gz' under Linux), the tar ball gives a number of files hanging from a subdirectory called xPand/. This directory must be placed at  (or linked from by typing in the directory containing the xPand directory 
'ln -s xPand MathematicaDirectoryForExternalApplications') one of the places 
Mathematica prepares for external applications:

Linux:

   - system-wide installation (requires root priviledges):

        /usr/share/Mathematica/Applications/

          or

        /usr/local/Wolfram/Mathematica/7.0/AddOns/Applications/

   - single-user installation:

        $HOME/.Mathematica/Applications/

Mac OS:

   - system-wide installation (requires root priviledges):

        /Library/Mathematica/Applications/

          or

        /Applications/Mathematica 7.0.app/AddOns/Applications/

   - single-user installation:

        /Users/<user>/Library/Mathematica/Applications/

MSWindows:

   - system-wide installation:

        C:\Documents and settings\All Users\Application data\Mathematica\Applications\

          or

        C:\Program Files\Wolfram Research\Mathematica\7.0\AddOns\Applications\

   - single-user installation:

        C:\Documents and settings\<user>\Application Data\Mathematica\Applications\

Beware that in Windows these directories might be hidden!

Documentation:
**************

To load the CMB documentation go to `Add-Ons and Packages' in the Documentation
Center (versions 6.0 of Mathematica at least required).

Then the packages can be loaded using unix style

        <<xPand/xPand.m
	

or Mathematica style

        <<xPand`xPand`
